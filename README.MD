## üéØ Problem-Solving Strategies

### Simplify First

> **George P√≥lya:** _"If you can't solve a problem, then there is an easier problem you can solve: find it."_

**üí° Prompt:**

```
I am struggling to understand [Topic]. Create a strictly simpler version of this problem
that retains the core concept, help me solve that, and then bridge it back to the original.
```

**üí° Alternative Prompt:**

```
I am stuck on [Problem]. Don't simplify it for me yet. Instead, ask me 3 guiding
questions that help me strip away constraints until I arrive at the 'Toy Version'
of this problem myself. Once I solve the Toy Version, guide me in adding the
constraints back one by one.
```

---

### Multiple Approaches

> **George P√≥lya:** _"It is better to solve one problem five different ways, than to solve five problems one way."_

**üí° Prompt:**

```
Provide 5 distinct methods to prove or solve [Problem/Theorem].
```

**üí° Alternative Prompt:**

```
Solve [Problem] using three radically different paradigms (e.g., Geometric,
Algebraic, and Probabilistic). For each solution, assign a 'Cognitive Cost' score
(how hard is it to set up?) and an 'Insight Score' (how much does it reveal about the nature of the problem?).
```

---

### Decomposition

> **Ren√© Descartes:** _"Divide each difficulty into as many parts as is feasible and necessary to resolve it."_

**üí° Prompt:**

```
Decompose [Complex Theorem/concept] into a hierarchical list of atomic prerequisites
I need to learn first.
```

---

### The Rising Sea Method

> **Alexander Grothendieck:** _"Don't try to crack the nut with a hammer; let the nut soak in water until it opens naturally."_

**üí° Prompt:**

```
Instead of giving me the brute-force solution to [Problem], teach me the general theory
and context surrounding it so the solution feels obvious.
```

**üí° Alternative Prompt:**

```
I have a specific hard problem: [Problem]. Do not solve it yet. Let's use Grothendieck's
'Rising Sea' method. Let's build up the general theory and definitions surrounding this
problem so thoroughly that the solution eventually becomes trivial/obvious. Start filling
the 'sea' of theory.
```

---

### Inversion Principle

> **Carl Gustav Jacobi:** _"Invert, always invert."_

**üí° Prompt:**

```
Explain [Function/Operation] by looking at its inverse. What does working backward tell
us about the forward process?
```

---

### Problem Formulation

> **David Hilbert:** _"A perfect formulation of a problem is already half its solution."_

**üí° Prompt:**

```
Rewrite the following problem in clearer, deeper formulations and show how each
formulation hints at a method of solution: [Problem].
```

---

### Reflection & Improvement

> **George P√≥lya:** _"Look back on your work. How could it be better?"_

**üí° Prompt:**

```
I have found the solution to this problem: [Solution]. Now, facilitate a 'Look Back' phase.
Critique my method. Was it the most efficient way? Is there a more elegant approach?
How can we generalize this result to a broader class of problems?
```

---

### Working Backwards

> **Carl Friedrich Gauss:** _"I have had my results for a long time: but I do not yet know how I am to arrive at them."_

**üí° Prompt:**

```
I intuitively believe the answer to [Problem] is [Answer], but I can't prove it.
Act as Gauss. Help me work backwards from this result. What conditions must be true
for this result to hold? Let's construct the bridge from the destination back to the start.
```

## üî¨ Physics & Mathematics

### Unreasonable Effectiveness

> **Eugene Wigner:** _"The unreasonable effectiveness of mathematics in the natural sciences."_

**üí° Prompt:**

```
Why does [Abstract Math Concept] work so well to describe [Physical Phenomenon]?
Explain the link.
```

**üí° Alternative Prompt:**

```
Inspire me with the 'Miracle' of [Concept]. Why is it amazing that this math works?
```

---

### Model Limitations

> **Werner Heisenberg:** _"The existing scientific concepts cover always only a very limited part of reality, and the other part that has not yet been understood is infinite."_

**üí° Prompt:**

```
What are the current mathematical limitations in describing [physical phenomenon]?
```

---

### Models vs. Reality

> **Niels Bohr:** _"Physics is not about how the world is, but about what we can say about the world."_

**üí° Prompt:**

```
Explain the limitations of the model [Model Name] vs. the reality it attempts to describe.
```

**üí° Alternative Prompt:**

```
Explain [Physical Law] as an optimization problem. What quantity is the universe trying
to minimize or maximize here (e.g., Action, Energy, Time)?
```

---

### Paradigm Shifts

> **Max Planck:** _"New scientific truth does not triumph by convincing its opponents... but rather because its opponents eventually die."_

**üí° Prompt:**

```
Contrast the Classical view of [Topic] with the Modern view, and explain why the shift happened.
```

## üèóÔ∏è Abstraction & Structure

### Formal Systems

> **Bertrand Russell:** _"Mathematics may be defined as the subject in which we never know what we are talking about, nor whether what we are saying is true."_

**üí° Prompt:**

```
Teach me [Topic] as a formal axiomatic system, divorced from any real-world meaning.
```

---

### Relational Thinking

> **Henri Poincar√©:** _"Mathematicians do not deal in objects, but in relations between objects; thus, they are free to replace some objects by others so long as the relations remain unchanged."_

**üí° Prompt:**

```
Help me map the relations in this problem, ignoring the specific objects for a moment.
Create a schematic diagram of how the components relate. Then, let's substitute simpler
objects into this relational map to see if the logic holds.
```

---

### Power of Abstraction

> **Alfred North Whitehead:** _"The utmost abstractions are the true weapons with which to control our thought of concrete fact."_

**üí° Prompt:**

```
Generalize this specific problem [Problem] into a more abstract class of problems,
then solve the general case.
```

---

### From Arithmetic to Algebra

> **Emmy Noether:** _"I received the decisive impulse to study abstract algebra from an arithmetical viewpoint."_

**üí° Prompt:**

```
Start with a simple arithmetic example of [Concept]. Now, generalize it step-by-step
until we arrive at the abstract algebraic definition. Show me the path from numbers
to symbols.
```

---

### Analogies Between Analogies

> **Stefan Banach:** _"Good mathematicians see analogies between theorems or theories. The very best see analogies between analogies."_

**üí° Prompt:**

```
What is the 'meta-analogy' that links [Field A] and [Field B]?
```

> **Gian-Carlo Rota:** _"Mathematics is the study of analogies between analogies."_

**üí° Prompt:**

```
I see an analogy between A and B. Is there a similar analogy between C and D?
And is there a 'master analogy' that connects the A-B pair to the C-D pair?
Help me climb the ladder of abstraction.
```

---

### Category Theory

> **Saunders Mac Lane:** _(On Category Theory)_ _"General abstract nonsense."_ _(Affectionate term)_

**üí° Prompt:**

```
Define [Mathematical Object] strictly via its Universal Property.
Don't look 'inside' the object at its elements. Describe it solely by how it
relates to other objects in its category (inputs and outputs). Draw the
commutative diagram using text-based art (ASCII).
```

**üí° Alternative Prompt:**

```
Don't tell me what the elements of this set are. Tell me about the arrows (morphisms)
between them. Define the object solely by its relationships to other objects.
```

---

### Forgetting Details

> **John Baez:** _"A great advance comes from forgetting unnecessary details."_

**üí° Prompt:**

```
Strip away the irrelevant details and show the core categorical essence: [Concept].
```

### Essential Structure

> **Emmy Noether(paraphrased):** _"Emmy Noether's creative power was directed... towards the clarification of mathematical structures... leaving all unnecessary entities... aside."_

**üí° Prompt:**

```
Take [Theorem]. Strip away all coordinate systems, specific numbers, and arbitrary choices.
What is the absolute 'essential' core structure that remains? Re-state the theorem using
only these essentials.
```

---

### Invariance

> **Emmy Noether(paraphrased):** _"All problems in mathematics are invariant under suitable transformations."_

**üí° Prompt:**

```
Analyze [Physical Law/Equation]. Perform a 'Symmetry Stress Test'.
What happens if I translate it in time? Rotate it in space? Scale the units?
Which transformations leave the equation unchanged, and what Conserved Quantity
(via Noether's Theorem) corresponds to each symmetry?
```

---

### Building New Universes

> **Grothendieck(paraphrased):** _"To understand a problem, build a new universe where it becomes natural."_

**üí° Prompt:**

```
Construct the simplest universe/category where [Problem] becomes trivial.
```

---

### Mathematical Freedom

> **Georg Cantor:** _"The essence of mathematics lies in its freedom."_

**üí° Prompt:**

```
Invent a new mathematical structure with arbitrary rules (axioms) and let's explore
its consequences together.
```
